# Panelist UI kit (port 3001)

Panelists (dept heads) govern a registry through a 3-of-5 multisig. This kit covers the three primary tabs:

- **Proposals** — queue of enrollment, revocation, and panelist-replace proposals. Each row shows type pill, status pill, progress (2/3 etc.), and an Approve action.
- **Pending students** — students awaiting approval. Panelist pastes the student's IPFS CID and approves.
- **Panelists** — 5-slot directory.

Accent: indigo (`--panelist`).
